# THE CLEAVE · early exit

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **A SCANNER DARKLY**.

THE CLEAVE — not every token is worth the whole model. A live instrument of EARLY-EXIT / CALM (Confident Adaptive Language Modeling): each layer refines the guess, and the token cleaves out at the first depth whose confidence clears a gate τ. Drag τ and watch the trade in real time — a loose gate is cheap but breaks more answers, a strict gate is faithful but pays for depth it didn't need; the knee is the whole game. GREEN the exit rule and every number (avg exit depth, % compute saved, agreement, the whole τ-sweep) computed live in-browser · AMBER a toy calibrated 12-layer confidence field, agreement measured vs its own full-depth answer · RED a real model needs trained exit heads on a shared trunk and still pays KV 'state-copying' for later tokens. Sibling of [[the-draft]] and [[the-stitch]]. David Lee Wise (ROOT0), with AVAN.
